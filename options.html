Configurações de Prazos e Feriados

Prazos para Telas Triar/Tratar e Tramitação ? Ajuste os cálculos de datas para diferentes ações. Valores negativos indicam dias ANTES do prazo de referência; valores positivos indicam dias DEPOIS. Os placeholders (ex: {PRAZO}) que usam estas datas são detalhados na aba "Modelos de Texto".

Configurações de Ajuste de Data (Fim de Semana e Feriados) ? Regras para ajustar datas calculadas que caem em fins de semana ou feriados. O ajuste de feriado é aplicado APÓS o ajuste de fim de semana, se a data ainda cair em um feriado.

Configuração de Feriados: ? Liste os feriados (DD/MM/AAAA) que devem ser desconsiderados nos cálculos de "dias úteis". Feriados são específicos por ano.

Editar Modelos de Texto

Personalize os textos para os assistentes. As "chaves" principais (ex: "Arquivar") são os assistentes. ? Dentro de cada assistente, defina opções de texto. Use os placeholders listados abaixo para inserir dados dinâmicos.
Ver Guia de Placeholders Disponíveis

Os placeholders são códigos especiais que a extensão substitui por informações reais da manifestação ou configurações. Use-os nos seus modelos de texto:

  • {datalimite} - Data limite original de atendimento da manifestação (ex: "dd/mm/aaaa"). Usado na funcionalidade "Melhoria de Tratar Manifestação" (botão de texto para prorrogação).
  • (NUP) - Número Único de Protocolo da manifestação (ex: "00123.000456/2024-01"). Usado pelo Assistente de Arquivamento.
  • {OUVIDORIA} - Nome da ouvidoria de destino selecionada. Usado pelo Assistente de Encaminhamento no texto para o solicitante.
  • ${numeroManifestacao} - Número da manifestação (NUP). Usado pelo Assistente de Encaminhamento no texto para o solicitante.
  • {PRAZO} - Data calculada para tramitação interna (ex: "dd/mm/aaaa"). Configurada na aba "Prazos e Feriados". Usado pelo Assistente de Tramitação.
  • {SECRETARIA} - Tag da unidade/secretaria de destino da manifestação. Usado pelo Assistente de Tramitação.
Ver Exemplo da Estrutura JSON para Modelos 
{
  "mensagens": {
    "prorrogacao": "Prezados, informamos que o prazo para resposta à sua manifestação foi estendido até {datalimite} para melhor análise."
  },
  "Arquivar": {
    "Duplicidade": "Manifestação (NUP) arquivada por duplicidade com o protocolo X."
  },
  "Encaminhar": {
    "Encaminhamento Geral": {
      "destinatario": "Encaminhamos para análise e providências.",
      "solicitante": "Sua manifestação ${numeroManifestacao} foi encaminhada para {OUVIDORIA}."
    }
  },
  "Prorrogar": {
    "Análise Complexa": "Prorrogação solicitada devido à complexidade da análise."
  },
  "Tramitar": {
    "Solicitação de Resposta": "À {SECRETARIA}, solicitamos análise e proposta de resposta até {PRAZO}."
  }
}

Atenção: JSON inválido pode quebrar os assistentes. Valide a estrutura antes de salvar. Faça backup se realizar grandes alterações.

Editar Pontos Focais

Configure os Pontos Focais para o "Assistente de Tramitação". A "chave" é a sigla da Unidade (ex: "GABINETE"), o "valor" é uma lista com os nomes dos destinatários.
Ver Exemplo da Estrutura JSON para Pontos Focais 
{
  "OUV-ABC": [
    "Fulano de Tal da Silva (OUV-ABC)",
    "Ciclana de Souza (OUV-ABC)"
  ],
  "SECEX": [
    "Beltrano dos Santos (SECEX)"
  ]
}

Atenção: JSON inválido pode impedir o funcionamento do Assistente de Tramitação. Valide e faça backup antes de salvar.

Configuração JSON Completa (Avançado) ? Edição direta do arquivo de configuração global do Neuron. Use com cautela. Todas as configurações das outras abas estão refletidas aqui.

Atenção: Erros na estrutura deste JSON podem causar falhas na extensão ou perda de configurações. Recomendado apenas para usuários avançados. Faça backup!

Importar/Exportar Configurações

Exportar Configurações ? Cria um arquivo JSON com todas as suas configurações atuais do Neuron. Útil para backup ou para transferir para outra instalação.

Importar Configurações ? Carrega configurações de um arquivo JSON. ATENÇÃO: Isso substituirá TODAS as suas configurações atuais pelas do arquivo importado.

Salva modificações das abas "Geral", "Prazos & Feriados", "Modelos de Texto" e "Pontos Focais". ? Este botão aplica as mudanças feitas na UI das abas principais. A aba "Configuração JSON" possui botões próprios para salvar seu conteúdo específico. Alterações feitas diretamente nos editores JSON das abas "Modelos de Texto" e "Pontos Focais" também são salvas por este botão.
